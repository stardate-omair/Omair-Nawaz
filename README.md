# Omair Nawaz - Personal Portfolio

A modern, interactive personal resume website featuring a custom cursor distance tracker and constellation background animation.

## Tech Stack
-   **HTML5 & CSS3** (Custom properties, Flexbox/Grid)
-   **JavaScript** (ES6+, Canvas API)
-   **No Frameworks**: Pure vanilla implementation for performance and simplicity.

## How to Run Locally

You can run this project locally using Python (pre-installed on macOS/Linux) or Node.js.

### Option 1: Python (Recommended)
Open your terminal in this directory and run:

```bash
# Python 3
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

### Option 2: Live Server (VS Code)
If you use VS Code, install the "Live Server" extension and click "Go Live" in the bottom right corner.

### Option 3: Node.js
If you have Node.js installed:

```bash
npx serve .
```

## Deployment

To put this website on the public internet, I recommend **GitHub Pages** or **Vercel**.

### GitHub Pages (Free & Easy)
1.  Push this code to a GitHub repository.
2.  Go to Repository Settings > Pages.
3.  Select the `main` branch as the source.
4.  Your site will be live at `https://yourusername.github.io/repo-name`.

### Vercel / Netlify
1.  Connect your GitHub account.
2.  Import the repository.
3.  It will automatically detect the static site and deploy it.
